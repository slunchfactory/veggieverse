# 레시피 이미지 폴더

## 📁 구조

```
recipe/
├── thumbnails/        # 레시피 썸네일 이미지 (카드용)
│   ├── recipe-1.jpg
│   ├── recipe-2.jpg
│   ├── recipe-101.jpg
│   └── ...
└── details/           # 레시피 상세 이미지 (히어로 이미지)
    ├── recipe-1.jpg
    ├── recipe-2.jpg
    ├── recipe-101.jpg
    └── ...
```

## 📝 이미지 파일명 규칙

- 썸네일: `recipe-{레시피ID}.jpg` 또는 `recipe-{레시피ID}.png`
- 상세 이미지: `recipe-{레시피ID}.jpg` 또는 `recipe-{레시피ID}.png`

### 명예의 전당 (Top 20)

### 원형 캐러셀
- ID 1-20

## 💡 사용 방법

1. 레시피 썸네일 이미지를 `recipe/thumbnails/` 폴더에 추가
2. 레시피 상세 이미지를 `recipe/details/` 폴더에 추가
3. 파일명을 `recipe-{ID}.jpg` 형식으로 저장
4. 이미지가 없으면 자동으로 재료 아이콘이 표시됩니다

## 📐 이미지 권장 사이즈

- **썸네일**: 500x500px 이상 (정사각형 권장)
- **상세 이미지**: 1200x800px 이상 (16:10 비율 권장)
